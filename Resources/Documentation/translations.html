Translating Aria Maestosa

Aria Maestosa uses Transifex for translations

If you are interested in helping translate Aria Maestosa, help is welcome!
Start by visiting the Transifex Aria Maestosa page.

  • If this is your first time on Transifex, you will need to login. You can create a new account, or conveniently you can use your Facebook, Twitter or Google account to log in.

  • On the langages page (see link above), start by seeing if a team already exists for the language you are interested in.

    • If the language exists, click on it and then click on the Join language translation button (1. in the image below). After you have been approved, you will be able to translate the Application Strings (2. in the image below) from the same page.


    • If no team for your language exists, press the Request language button or email me (see the Contact page for my email)


Your efforts to bring Aria Maestosa to new languages are highly appreciated!
